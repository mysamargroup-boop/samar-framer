# **App Name**: Eternal Frame Photography

## Core Features:

- Dynamic Portfolio Galleries: Centralized system for managing and displaying categorized photography work in engaging full-screen sliders and a filterable masonry layout.
- Booking & Contact Management: A comprehensive system allowing clients to easily book sessions or submit detailed inquiries via a dedicated form, with backend storage for lead management.
- Service Packages & Pricing Display: Clearly present detailed photography packages with pricing (in INR), custom add-on options, and descriptions on a dedicated page.
- Studio About & Testimonials: Showcase the studio's story, photographer profiles, achievements, and client testimonials with associated images in elegantly designed sections.
- Optimized Media Delivery: Implement efficient image optimization and lazy loading techniques to ensure fast loading times for high-resolution images across the entire website.
- Responsive Layout & Navigation: A fluid, mobile-responsive design with a sticky navigation bar for universal accessibility and an elegant footer containing essential information and social links.
- AI-Enhanced Portfolio Suggestions Tool: An AI tool to intelligently suggest and highlight specific portfolio images or categories based on user interests or queries, offering a more personalized browsing experience.

## Style Guidelines:

- Primary color: Deep Amethyst (#6A508B) for a luxurious and cinematic feel, evoking creativity and sophistication against dark backgrounds.
- Background color: Nearly black charcoal (#1C191D) providing a premium, immersive canvas that makes imagery and other elements pop.
- Accent color: Vibrant Blue-Violet (#6888EC) used sparingly for calls to action and highlights, adding a contemporary and energetic contrast.
- Headline font: 'Belleza' (humanist sans-serif) for its elegant, fashion-forward, and artistic aesthetic. Body font: 'Alegreya' (humanist serif) for refined readability and intellectual charm.
- Elegant, minimalist line-art or subtly filled icons will be used for features and contact points, maintaining a sophisticated and clean aesthetic suitable for a luxury brand.
- Utilize full-width, immersive photographic displays and cinematic content blocks, with a sticky header for navigation. Portfolio sections will feature a masonry grid layout to showcase diversity in work, providing a high-end visual experience.
- Subtle and smooth animations will be applied to page transitions and content loading. Refined hover effects on interactive elements and controlled parallax scrolling for select hero sections will enhance visual depth and a premium user experience.
